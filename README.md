### рҹҗ§ &nbsp; Linux / VPS

<br/>

**Step 1 вҖ” Install Node.js, FFmpeg & Git**
```bash
sudo apt update && sudo apt upgrade -y
sudo apt install git ffmpeg -y

# Install Node.js 20
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install nodejs -y

# Verify
node -v    # v20.x.x
npm -v     # 10.x.x
ffmpeg -version
```

<br/>

**Step 2 вҖ” Clone the project**
```bash
git clone https://github.com/xnodesdevelopers/GARFIELD-V11-CORE.git
cd GARFIELD-V11-CORE
```

<br/>

**Step 3 вҖ” Install dependencies**
```bash
npm install
npx playwright install chromium
```

<br/>

**Step 4 вҖ” Configure**
```bash
nano config.js
```
```js
OWNER_NUMBER:   '94711234567',  // your number without +
PREFIX:         '.',
MODE:           'public',
GEMINI_API_KEY: 'your_key_here', // optional вҖ” get from aistudio.google.com
```

<br/>

**Step 5 вҖ” Add menu video**
```bash
# put your menuvideo.mp4 inside lib/ folder
ls lib/menuvideo.mp4   # must exist
```

<br/>

**Step 6 вҖ” First launch & scan QR**
```bash
node index.js
```
> рҹ“ұ QR code appears вҶ’ Open WhatsApp вҶ’ **Linked Devices** вҶ’ **Link a Device** вҶ’ Scan
> After scan press `Ctrl+C`

<br/>

**Step 7 вҖ” Install PM2 & start bot**
```bash
npm install -g pm2
npm start
```
